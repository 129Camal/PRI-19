Abstract:

Este trabalho tinha por objetivo criar um servidor em NodeJS que recebesse pedidos através da porta 4002 e respondesse com a página html correta.

A linguagem utilizada para realizar o servidor foi o Javascript.

Associado ao servidor estão páginas html que representam o Arquivo Musical de uma banda. Em que temos o índice e uma página html por cada obra.

Relativamente aquilo que foi criado nas aulas apenas alterei as âncoras das páginas html, para não navegarmos nos ficheiros presentes na máquina, mas sim, realizar os pedidos diretamente ao servidor, este analisando-os e respondendo conforme o desejado pelo utilizador.

Deliverables:

  • Servidor
  • Ficheiro XSL com os templates para geração das páginas html
  • Ficheiro XML com toda a informação sobre o Arquivo Musical
  • Índice do Arquivo Musical
  • Exemplo de uma obra do Arquivo Musical
