Abstract:

Este trabalho tinha por objetivo criar um projeto em NodeJS que recebesse pedidos através da porta 5000 e respondesse com a página html correta e as suas necessidades sobre o CSS.

A linguagem utilizada para realizar o servidor foi o Javascript.

Associado ao servidor, estão vários ficheiros em Pug que são templates para a formação das páginas html através dos ficheiros fornecidos pelo professor em Json. Para criar o índice das obras, foi criada uma função que itera os ficheiros Json e extrai o título da obra e o nome do ficheiro e cria um ficheiro index.json.

Deliverables:

  • Servidor em NodeJS;
  • Package.json do projeto;
  • Layout Base em Pug;
  • Template em Pug referente a uma obra;
  • Template em Pug referente ao indice das obras;
  • Exemplo de obra em Json;
