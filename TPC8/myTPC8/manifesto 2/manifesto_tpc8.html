Abstract:

Este trabalho têm como objetivo criar um website que o envio e registo de ficheiros. Para isso foi criado um servidor em NodeJS e vários templates em Pug para representar o formulário e a informação dos ficheiros para lá carregados.

Foi também criado um script em JQUERY utilizando AJAX que trata de atualizar o cliente quando este introduz informação, sem fazer pedidos ao servidor de novas páginas.

Contudo, não foi possivel para mim resolver todo o enunciado.

Foram surgindo vários problemas para a qual não consegui encontrar solução. Sendo o principal, o envio do ficheiro utilizando AJAX para ser guardado pelo servidor.

Contudo, este projeto, envia para o servidor informação sobre a descrição e sobre o ficheiro para ser guardado num ficheiro JSON que serve de base de dados.

Deliverables:

  • App.js;
  • Package.json do projeto;
  • Layout Base em Pug;
  • Template para inserir elementos na tabela na página principal;
  • Template em Pug referente à página principal;
  • Ficheiro que trata dos pedidos recebidos pelo servidor;
  • Ficheiro referente ao JQUERY e AJAX no cliente;
