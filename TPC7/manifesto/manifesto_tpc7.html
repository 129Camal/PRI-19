Abstract:

Este trabalho têm como objetivo criar um website que o envio e registo de ficheiros. Para isso foi criado um servidor em NodeJS e vários templates em Pug para representar o formulário e a informação dos ficheiros para lá carregados.

Após enviar os ficheiros, o servidor guarda os dados do ficheiro num ficheiro JSON e o próprio ficheiro numa pasta. Esse ficheiro JSON é lido no template do formulário mostrando assim os ficheiros carregados anteriormente com possibilidade de os mostrar no browser.

Para além disso, é de salientar que este servidor consegue mostrar imagens e ficheiros pdf.

Deliverables:

  • Servidor em NodeJS;
  • Package.json do projeto;
  • Layout Base em Pug;
  • Template para apresentar o erro;
  • Template em Pug referente ao formulário para enviar o ficheiro;
  • Template em Pug referente à pagina após envio do ficheiro;
